# Сайт мануфактуры изделий из кожи «АРТ-Кожа»

Сайт разработан на стеке Python 3 + Flask + SQLite + JavaScript.

## Возможности
- 7 основных страниц контента + карта сайта + страница 404 с автоперенаправлением.
- Два стиля оформления: стандартный и версия для слабовидящих (ГОСТ Р 52872-2012):
  4 цветовые схемы, 3 размера шрифта, отключение изображений.
- Компоненты: баннеры, контакты с формой, система обмена сообщениями,
  лента новостей, поиск по сайту.
- Учётные записи (роли: сотрудник admin, покупатель client), регистрация,
  личный кабинет, сценарий создания пользователя сотрудником.
- Каталог (8 изделий), новости (6), статьи (6), услуги (5).

## Запуск локально
```bash
python -m venv venv
# Windows: venv\Scripts\activate   |  Linux/Mac: source venv/bin/activate
pip install -r requirements.txt
python init_db.py        # создаёт и наполняет database.db
python app.py            # сайт на http://127.0.0.1:5000
```

## Развертывание на хостинге (пример для PythonAnywhere)
1. Загрузить проект, создать virtualenv, `pip install -r requirements.txt`.
2. Выполнить `python init_db.py`.
3. В разделе Web указать WSGI: `from app import app as application`.
